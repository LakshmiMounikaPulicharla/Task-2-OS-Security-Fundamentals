# Task 2 – Operating System Security Fundamentals (Linux & Windows)

## Description
This repository contains Task 2 of the Cyber Security Internship.  
The task mainly focused on understanding basic operating system security concepts using Linux and Windows.

## Topics Covered
- User accounts and access control
- File permissions in Linux (ls -l, chmod, chown)
- Administrator vs standard user privileges
- Firewall configuration (UFW and Windows Firewall)
- Running processes and services
- Disabling unnecessary services
- Best OS hardening practices

## Files Included
- Task2.pdf – Documentation of OS security fundamentals

## Tools Used
- Ubuntu Linux / Windows OS
- Virtual Machine
- Built-in firewall tools

## Outcome
This task helped me understand the importance of operating system security. I learned how user accounts, file permissions, firewalls, and system services play a role in protecting the system. It also helped me understand how disabling unnecessary services and following OS hardening practices can reduce security risks.
